# ☕ BREWHA Documentation

## Overview
BREWHA is a web-based ecommerce and point-of-sale (POS) system developed for Brewha located inside CHMSU Talisay Campus. The system aims to improve ordering, transaction management, and administrative monitoring through a digital platform.

---

## Features
- Customer Ordering System
- Login and Signup Portal
- Point-of-Sale (POS) System
- Admin Dashboard
- Order Management
- Sales Monitoring
- User Authentication
- Web-Based Interface

---

## Project Purpose
The BREWHA system was developed as an academic and practical project to modernize business operations and provide a more efficient customer and management experience.

---

## System Modules
### Customer Module
- Browse menu
- Place orders
- Account login/signup
- View order status

### POS Module
- Process transactions
- Generate receipts
- Manage orders

### Admin Module
- Monitor sales
- Manage products
- Manage users
- View reports and analytics
